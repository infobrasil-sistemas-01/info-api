O Coração da Integração

Os dados do seu ERP, agora em qualquer lugar.

A InfoAPI atua como uma camada de inteligência e segurança entre o seu banco de dados Retaguarda e o mundo externo. Esqueça integrações complexas via arquivos ou acessos diretos inseguros.

  • Automação: Conecte ferramentas como n8n, Zapier ou Make para automatizar fluxos.
  • IA & Código: Use assistentes como Claude Code ou GitHub Copilot para ler e agir sobre dados reais.
  • Ecossistema: Alimente e-commerces, aplicativos móveis e Dashboards de BI sem latência.

Funcionalidades da API

Nossa arquitetura garante a estabilidade do seu banco de dados através de limites inteligentes e endpoints otimizados para todas as operações do seu ERP.

Produtos & Estoque

Controle completo de catálogo, variações de grade, tabelas de preço e saldos de estoque.

GET /products
Listar/Pesquisar produtos no catálogo
GET /products/id/:id
Obter detalhes de produto por ID
GET /products/barcode/:barcode
Buscar produto por código de barras
GET /products/groups
Listar grupos de produtos
POST /products/groups
Cadastrar novo grupo de produtos
PATCH /products/groups/:id
Atualizar grupo de produtos existente
GET /products/brands
Listar marcas de produtos
POST /products/brands
Cadastrar nova marca de produtos
PATCH /products/brands/:id
Atualizar marca de produtos existente

Vendas & Pedidos

Integração bidirecional de pedidos de venda diretamente no fluxo de faturamento do ERP.

POST /orders
Registrar novo pedido de venda
GET /orders
Listar histórico de pedidos
GET /orders/:id
Obter detalhes de pedido por ID
GET /order-items
Listar itens faturados ou vendidos
POST /orders/:id/receipt
Gerar cupom fiscal (NFC-e/SAT)

Clientes & Contatos

Sincronização de dados de clientes, endereços de entrega e históricos comerciais.

GET /clients
Listar/Pesquisar clientes cadastrados
GET /clients/:id
Obter ficha detalhada do cliente por ID
POST /clients
Cadastrar novo cliente no ERP
PATCH /clients/:id
Atualizar dados cadastrais de cliente

Financeiro & Contas

Acompanhamento de fluxo de caixa, títulos a pagar/receber e formas de pagamento.

GET /account-payable
Listar contas a pagar do período
GET /account-payable/id/:id
Obter detalhes de conta a pagar por ID
GET /account-receivable
Listar contas a receber (títulos pendentes)
GET /payment-methods
Listar métodos de pagamento aceitos (cartões)
GET /payment-plans
Listar planos e prazos de pagamento

Funcionários & Cargos

Gestão de vendedores, atendentes, técnicos e controle de comissões operacionais.

GET /employees
Listar/Pesquisar equipe de funcionários
GET /employees/:id
Obter detalhes de funcionário por ID
GET /employee-roles
Listar cargos e funções de funcionários

Entregas & Rastreamento

Acompanhamento logístico, despacho de mercadorias e atualização de status de frete.

GET /deliveries
Listar/Pesquisar status das entregas
GET /deliveries/:id
Obter detalhes de uma entrega específica
POST /deliveries
Registrar nova entrega ou despacho logístico

Fornecedores & Serviços

Cadastro e consulta de fornecedores de mercadorias e prestadores de serviços de apoio.

GET /suppliers
Listar/Pesquisar fornecedores de estoque
GET /suppliers/:id
Obter detalhes de fornecedor por ID
GET /service-providers
Listar prestadores de serviços cadastrados
GET /service-providers/:id
Obter detalhes de prestador de serviços por ID

Lojas & Filiais

Consulta de dados corporativos de lojas e filiais vinculadas à conta do cliente.

GET /stores
Listar filiais operacionais do ERP
