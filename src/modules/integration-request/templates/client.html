Manual de Integração

Siga estes passos para conectar sua aplicação à InfoAPI em minutos.

1

Autenticação (Obter Token)

Toda comunicação é protegida. Primeiro, você deve trocar suas credenciais por um token de acesso.

POST /api/v1/auth/login
Authorization: Basic [USUARIO:SENHA em Base64]

O retorno conterá um access_token (válido por 1h) e um refresh_token.

2

Realizar Requisições

Com o token em mãos, adicione-o ao cabeçalho de todas as chamadas seguintes.

GET /api/v1/products
Authorization: Bearer [SEU_ACCESS_TOKEN]
3

Renovação Automática (Refresh)

Para evitar que sua integração pare de funcionar quando o token expirar, use o refresh_token.

Dica: Não peça para o usuário logar de novo. Use o refresh token para pegar um novo acesso silenciosamente.
POST /api/v1/auth/refresh
Body: { "refresh_token": "..." }

Manual para sua IA (Claude, ChatGPT, etc)

Copie o prompt abaixo e cole na sua IA favorita para gerar o código de integração instantaneamente:

"Preciso de um script em Python (ou sua linguagem) para integrar com a InfoAPI. 1. O login é em /api/v1/auth/login usando Basic Auth. 2. Salve o access_token e o refresh_token. 3. Faça um GET em /api/v1/products com o token. 4. Se receber erro 401, use o refresh_token em /api/v1/auth/refresh para renovar o acesso e tente novamente."

JWT (JSON Web Token)

O JWT é um padrão para transmissão segura de informações. Na nossa API, ele é o "crachá" que você usa após o login.

Stateless (Sem Sessão) Auto-contido Cabeçalho Authorization
Authorization: Bearer [SEU_TOKEN]

Autenticação Basic

Método usado exclusivamente na rota de Login para obter seu token JWT inicial. O cabeçalho é composto pela string usuario:senha codificada em Base64.

Exemplo em JavaScript:

const auth = btoa('user:pass');
console.log('Basic ' + auth);

Exemplo em Python:

import base64
auth = base64.b64encode(b'user:pass').decode()
print(f'Basic {auth}')
